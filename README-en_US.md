## Introduction

**ilus** [Estonian], means beauty. 

Fast, concise, and powerful blogging system developed in Golang

The function of this project is located on Blog, CMS and WeChat miniprogram backend. 

## Quick start

1. Install golang development environment [Golang v1.12.pkg](https://dl.google.com/go/go1.12.darwin-amd64.pkg)

2. Install beegoï¼š`go get github.com/astaxie/beego`

3. Install bee toolï¼š`go get github.com/beego/bee`

4. Install mysql driver for Golangï¼š`go get -u github.com/go-sql-driver/mysql`

5. Install redis module for Golangï¼š`go get github.com/gomodule/redigo/redis`

6. Clone code, and change [mysql] in `app.conf`ï¼ˆ for default `is_cache=false`, you can make it `true` if you need ï¼‰

7. Build a new schema in mysqlï¼Œschema name must bee same with what you code in `app.conf`
